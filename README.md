# Development

### Setup

1. Clone the project

```
git clone https://github.com/dgoldstein1/websiteAnalytics-frontend.git
cd websiteAnalytics-frontend
```

2. Build and up containers

```
docker-compose up -d --build
```

3. Navigate to http://localhost:3000/ in your browser.
